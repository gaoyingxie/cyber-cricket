**容器环境修复过程（重要教训）：**
- Debian 12 容器没有 sudo，需要手动安装
- /etc/apt/sources.list 初始不存在，导致所有 apt 操作失败
- /etc/sudoers.d/ 目录存在但主 sudoers 文件缺少 @includedir 指令，导致 node 用户 sudo 配置无效
- 解决：apt-get install sudo → echo "@includedir /etc/sudoers.d" >> /etc/sudoers → echo "node ALL=(ALL) NOPASSWD: ALL" >> /etc/sudoers.d/node
- 清华镜像 debian.tuna.tsinghua.edu.cn 在此容器内返回 403，换用官方 deb.debian.org
