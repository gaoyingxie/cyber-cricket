## 🎮 游戏简介

**赛博斗龙虾** 是一款roguelike循环养成游戏。你将操控一只从虾苗养成的龙虾，通过5轮**全自动策略战斗**逐步进化为究极龙虾！

**完全自动战斗**，只需观看龙虾自动战斗、积累技能、搭配Build，享受「赛博斗蛐蛐」的乐趣！

## 🕹️ 游戏模式

- **养虾模式** — 从Lv.1幼虫开始，通过5轮战斗养成究极龙虾，通关获得成虾代码
- **成虾导入** — 导入之前保存的成虾代码，挑战更高难度敌人

## ⚔️ 核心玩法

### 养成循环
```
选择敌人 → 自动战斗 → 偷学技能 → 掉落装备 → 升级进化
     ↑                                              ↓
     ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
              通关后可导出成虾代码
```

## 🛡️ 技能系统

> **全自动释放**，无需手动操作，纯粹观战！

### 主动技能 (17个)
重击、剧毒、病毒注入、数据入侵、电磁脉冲、护盾击破、处决、灵魂吸取、撕裂、混乱、封印、冰冻、沉默、防御、能量护盾、自我修复、镜反

### 被动技能 (9个)
必杀(暴击)、吸血、反击、神佑(复活)、护甲、荆棘、狂暴、善恶有报、愤怒、连击

## 📦 装备系统

## 📈 进化时机

- **Lv.5（第4轮结束）** — 幼虾→战斗虾，获得属性加成+随机技能

## 🎯 战斗机制

- **DOT伤害** — 中毒、流血每回合结算
- **控制效果** — 眩晕、封印、沉默

- **暴击与连击** — 被动技能触发
