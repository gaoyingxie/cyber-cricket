# 宝宝成长记录 - SPEC.md

## 1. Project Overview

- **Name**: 宝宝成长记录 (Baby Log)
- **Type**: H5 单页应用（移动端优先）
- **Core Functionality**: 记录宝宝日常（大小便、喝奶），支持统计展示
- **Target Users**: 新手爸妈，快速记录宝宝状态

## 2. 技术方案

- **前端**: 纯HTML/CSS/JS，单文件
- **存储**: localStorage（本地）
- **备份**: JSON 导出/导入

## 3. 数据模型

```json
{
  "id": "时间戳字符串",
  "type": "大便 | 小便 | 喝奶-奶粉 | 喝奶-母乳",
  "time": "ISO时间字符串"
}
```

## 4. 功能模块

### 4.1 单页布局（无Tab）

- **快速记录区**: 2x2 网格四个大按钮
- **今日统计**: 三个数字卡片（💩🚼🍼）
- **最近记录**: 今日记录列表，最新在上
- **设置菜单**: 导出/导入备份

### 4.2 记录按钮

- 💩 大便
- 🚼 小便
- 🍼 喝奶-奶粉
- 🤱 喝奶-母乳

### 4.3 统计

- 仅显示今日统计（最实用）
- 三个类型各多少次

### 4.4 历史

- 显示今日记录（倒序）
- 如果今日无记录，显示最近几条

### 4.5 备份

- 导出: 生成 JSON 文件
- 导入: 合并 JSON 文件（去重）
