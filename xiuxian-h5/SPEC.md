## 一、游戏概述与定位

### 1.1 游戏简介

| 项目 | 内容 |
|------|------|
| 游戏名称 | 凡人修仙传·人界篇 |
| 类型 | H5放置+轻度策略RPG |
| 平台 | 浏览器（移动端优先，支持PC） |
| 目标用户 | 修仙小说粉丝、放置类游戏爱好者 |
| 核心体验 | 境界突破的爽感、资源匮乏的紧迫感、功法收集的成就感 |

### 1.2 世界观简述

韩立从七玄门踏上修仙路，一路历经黄枫谷、掩月宗、灵兽山等势力，在人界各地冒险，突破炼气→筑基→结丹→元婴→化神，最终飞升灵界。人界篇是小说的核心舞台，讲述韩立如何在弱肉强食的修仙界存活并成长。

### 1.3 美术风格建议

- **主色调**：水墨丹青 + 古风暖色（赭石/青绿/墨黑）
- **UI风格**：古典卷轴 + 玉简令牌元素
- **战斗特效**：剑气纵横、灵气波动（粒子效果为主，卡通渲染）
- **免费素材优先**：使用开源古风素材（Kenney RPG pack、OpenGameArt），自研配色和UI布局

## 二、核心玩法循环

### 2.1 主循环

```
[挂机修炼] → [自动战斗] → [掉落资源] → [突破境界] → [解锁功法/灵兽] → [挑战更高副本]
     ↑                                                                      ↓
     ←────────────── [资源不足/等待] ←─────────────────────────────────────
```

### 2.2 每日玩家行为

| 阶段 | 时长 | 行为 |
|------|------|------|
| 上线收菜 | 5min | 收取挂机收益，升级功法，整理灵兽 |
| 挑战副本 | 10min | 手动通关1-2个新副本 |
| 定时收取 | 随时 | 收取离线收益，循环挂机 |

### 2.3 放置收益

- 基础挂机：每5分钟产出一批灵石+经验
- 离线收益：最长24小时，离线时间越长衰减越多
- 加速券：付费/活动道具，可缩短挂机间隔

- 每个功法最高10层

- **剑修路线**：青元剑诀 + 剑系被动 → 高攻低防

- **吸血刺客**：血影秘术 + 暴击被动 → 高爆发

## 七、战斗系统

### 7.1 战斗模式

**自动回合制战斗**

- 速度决定出手顺序
- 每回合：普攻 → 技能1 → 技能2 → 普攻（循环）
- 灵兽技能：每3回合触发一次

### 7.2 伤害公式

```
最终伤害 = 攻击力 × 技能倍率 × (1 - 防御减免) × 属性克制 × 暴击修正
```

- 防御减免上限：70%
- 暴击倍率：1.5x（可被功法提升至2x+）
- 属性克制：火>风>雷>土>水>火（循环）

### 7.3 战斗表现

- 战斗速度可调：1x / 2x / 3x
- 战斗跳过券：已通关的关卡可消耗跳过券直接结算
- 离线战斗：设置后自动连续战斗n场（需开启）
