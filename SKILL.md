# 赛博斗龙虾 - 技能公式文档

> v1.95

## 伤害公式

### 普通攻击
```
基础伤害 = max(1, floor(ATK × 1.0))
实际伤害 = floor(基础伤害 × (1 - DEF / (基础伤害 + 100)))
```

### 防御减伤
- 护盾：每层吸收 1 点伤害
- 防御（主动技能）：本回合受伤降低 60%
- 护甲（被动）：受到伤害永久降低 15%

## 数值计算示例

假设：ATK = 10，DEF = 4，敌人 maxHP = 200

### 普通攻击
```
基础 = floor(10 × 1.0) = 10
实际 = floor(10 × (1 - 4 / 110)) = floor(10 × 0.964) = 9
```

### 重击
```
基础 = floor(10 × 2.0) = 20
实际 = floor(20 × 0.964) = 19
```

### 剧毒（每回合）
```
直接伤害 = floor(10 × 0.6) = 6
DOT = floor(200 × 0.08) = 16
总计 = 6 + 16 = 22/回合
3 回合总计 ≈ 54
```

## 难度曲线

### 敌人属性（每轮）
```
生命 = floor((100 + round×15) × 难度系数)
攻击 = floor((7 + round×1.3) × 难度系数)
防御 = floor((3 + round×0.8) × 难度系数)
速度 = floor((5 + round×0.6) × 难度系数)
技能数 = round (+1 对第三个敌人)
```

### 难度系数
- 1-3 轮：0.6
- 4-5 轮：0.8
- 6+ 轮：1.0
